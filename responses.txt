commands in cmd
cd: Change the current directory.Example: cd C:\Users changes the current directory to "C:\Users".

dir: List files and directories in the current directory.
Example: dir lists the contents of the current directory.

mkdir: Create a new directory.
Example: mkdir MyFolder creates a new directory named "MyFolder".

rmdir: Remove a directory.
Example: rmdir MyFolder deletes the directory named "MyFolder".

copy: Copy files or directories.
Example: copy file.txt destination copies "file.txt" to the specified destination.

move: Move files or directories.
Example: move file.txt destination moves "file.txt" to the specified destination.

del: Delete a file.
Example: del file.txt deletes "file.txt".

ren: Rename a file or directory.
Example: ren oldname newname renames "oldname" to "newname".

ping: Send a network request to a specific IP address or domain.
Example: ping google.com sends a ping request to Google's servers.

ipconfig: Display the IP configuration of network interfaces.
Example: ipconfig shows information about network adapters, IP addresses, etc.

netstat: Display active network connections and listening ports.
Example: netstat -a shows all active connections and listening ports.

tasklist: Display a list of currently running processes.
Example: tasklist lists all running processes on your system.

taskkill: Terminate a running process.
Example: taskkill /IM process.exe terminates the process with the specified name.

systeminfo: Display detailed information about the computer's hardware and software.
Example: systeminfo provides system information like OS version, hardware details, etc.
